# Architecture Overview

This project uses Azure managed services to enforce secure-by-default configurations and reduce operational risk.

## Components
- Azure App Service (PaaS)
- Azure Entra ID
- Access Restrictions
- Azure Monitor

## Design Principles
- Identity-first security
- Least privilege
- Reduced public exposure
- Cost-aware engineering

## Production Expansion
- Private Endpoint
- WAF
- Centralized logging
